## Описание

Веб приложение для анализа файлов, содержащих статистику запросов и блокировок для баз данных PostgreSQL (pgstat и locks).

## Возможности

- Импорт данных напрямую с диска: через указания пути к директории или к zip архиву;
- Наглядное отображение запросов и блокировок;
	- Отображение всплывающих подсказок, основанных на официальной документации Postgres для статусов транзакций, типов блокировок, wait event. Подсказки, помимо основного описания, содержат советы по оптимизации конфигурации БД;
	- Удобное построение таймлайна с отображением активных запросов и кол-ва блокировок на основе загружаемых данных;
	- Подсветка синтаксиса и автоматическое форматирование Query по даблклику;
	- Автоматическая конвертация времени UTC;
	- Возможность выбрать смещение временной зоны для того, чтобы время на таймлайне и время в самих запросах было в одном часовом поясе;
	- Возможность фильтрации запросов по статусу, а также типу блокировки;
- Автоматическое объединение одинаковых блокирующих транзакций по типу блокировки;
- Анализ корневых блокировок и очереди ожидающих запросов;
- Поиск по содержимому с подсветкой совпадающей подстроки;
	- Автоматическая фильтрация данных на таймлайне с отображением кол-ва совпадений. *(Отображается только то время, которое содержит в себе совпадения)*;
- Автоматическое построение гистограммы за весь загруженный период для визуальной оценки ситуации с блокировками с учетом временных отрезков;
- Возможность переключения на светлую и темную тему;
- Отслеживание потребления ОЗУ программой в реальном времени;

## Системные требования

| Компонент          | Требования                                                                |
| ------------------ | ------------------------------------------------------------------------- |
| Процессор (CPU)    | Любой современный 2-ядерный процессор (от 1.5 ГГц)                        |
| ОЗУ | приложению достаточно 2 ГБ – 4 ГБ (Зависит от обьема импортируемых данных)       |
| ПО                 | Java 21                                                 |

## Конфигурация запуска

```bash
java -Xms256m -Xmx4g -XX:+UseStringDeduplication -jar StatAnalyzer-X.X.X.jar
```
